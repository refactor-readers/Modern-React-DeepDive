# 13.1 구글 라이트 하우읏
라이트하우스(Lighthouse)는 구글에서 만든 웹 페이지 성능 측정 도구로, 접근성, SEO 등 다양한 지표를 제공

## 13.1.1 구글 라이트하우스란?
- 특징 : 오픈 소스로 운영되며, 단순 성능 외에도 웹 표준 준수 여부를 확인하기에 적합
- 측정 항목
    - Performance: 웹페이지의 속도와 관련된 지표
    - Accessibility : 장애인 및 노약자를 위한 웹 접근성 준수 여부
    - Best Practices: 웹 보안 및 최신 웹 표준 권장 사항 준수 여부
    - SEO : 검색 엔진 최적화 점수
    - PWA : 프로그래시브 웹 앱 요건 충족 여부

# 13.2 WebPageTest
전 세계 다양한 서버 위치와 네트워크 환경에서 웹사이트 성능을 정밀하게 테스트할 수 있는 도구

## 13.2.1 WebPageTest란?
- 주요 기능
    - Waterfall 차트 : 리소스 로드 순서와 시간을 시각적으로 제공
    - Web Vitals : LCP, CLS, TBT 등 핵심 지표 분석
    - 비디오 녹화 : 페이지가 그려지는 과정을 프레임별로 확인 가능

# 13.3 크롬 개발자 도구
가장 실무에서 자주 사용되며, 실시간으로 메모리 누수나 네트워크 병목 현상을 진단할 수 있음

## 13.3.1 Performance 탭
- 기능 : 웹페이지 로딩부터 사용자 인터랙션까지의 모든 과정을 타임라인으로 기록
- CPU 프로파일링 : 어떤 자바스크립트 함수가 메인 스레드를 오래 점유하는지 확인할 수 있음
- 네트워크/스크린샷 : 시간 흐름에 따른 화면 변화와 리소스 로드 시점을 대조 분석

## 13.3.2 Memory 탭
- 기능 : 자바스크립트 힙(Heap) 메모리 할당 상황을 확인하여 메모리 누수(Memory Leak)를 찾음
- 힙 스냅샷: 특정 시점의 메모리 사용 상태를 촬영하여 비교 분석
