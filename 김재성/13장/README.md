# 13장. 웹페이지의 성능을 측정하는 다양한 방법
성능 지표를 측정하고 분석하는 도구들입니다.

## 13.1 애플리케이션에서 확인하기
reportWebVitals 함수를 사용하여 콘솔에 지표를 출력하거나 분석 서버로 전송할 수 있습니다.

## 13.2 구글 라이트하우스 (Lighthouse)

### 13.2.1 탐색 모드: 페이지 접속부터 로딩 완료까지의 성능을 측정합니다.

### 13.2.2 기간 모드: 사용자가 특정 작업을 수행하는 기간 동안의 성능을 측정합니다.

### 13.2.3 스냅샷: 현재 페이지 상태를 기준으로 최적화 여부를 분석합니다.

## 13.3 WebPageTest
다양한 위치와 기기 환경에서 심도 있는 테스트를 제공하는 도구입니다.

Filmstrip: 0.1초 단위의 스크린샷을 통해 시각적 로딩 과정을 확인할 수 있습니다.

Web Vitals: LCP, CLS, TBT(Total Blocking Time) 등의 지표를 상세히 보여줍니다.

## 13.4 크롬 개발자 도구

### 13.4.1 성능 통계 (Performance Insights): 웹사이트 로딩 과정을 단계별로 녹화하고 이벤트별로 분석할 수 있습니다.

### 13.4.2 성능 (Performance): 메인 스레드의 작업, 스크립트 실행 시간 등을 정밀하게 시각화합니다. 긴 작업(Long Task)을 찾아내고 최적화하는 데 필수적입니다.
